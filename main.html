============================================= ____ ___ ____ _____ ____________ / __ \/ | / __ \/ ___// ____/ ____/ / /_/ / /| | / /_/ /\__ \/ __/ / / / ____/ ___ |/ _ _/___/ / /___/ /___ /_/ /_/ |_/_/ |_|/____/_____/\____/ ============================================= Task Manager ============================================= PARSEC Task Manager é uma ferramenta que permite a criação de tarefas pelo usuário. Ao serem concluídas, as tarefas recompensam o usuário com uma pontuação, usada para subir o nível do usuário dentro do sistema. OBS: Para completar o seu cadastro, preencha os campos em branco na aba PERFIL. Na aba MENSAGENS é possível trocar mensagens com outros usuários. Para isso, é necessário ter o código de identificação de quem você deseja contactar, inseri-lo no campo de destinatário, escrever a mensagem e ENVIAR. É nessa aba onde algumas das notificações sobre avanço de nível e tarefas concluídas, ou próximas da data limite aparecerão. Esperamos que essa ferramenta seja útil. Atenciosamente, Equipe PARSEC.
